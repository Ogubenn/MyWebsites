Jungle Jumper Game

My game is an endless runner game. It features a jungle-themed environment where the player controls a character who runs and jumps to avoid obstacles and collect power-ups.

ZigZag Game

My game is an endless physics-based single-player game. Players control a ball that rolls on a zigzagging path, and they must navigate it through narrow gaps and turns without falling off.

Pixel Adventure Game

My game is a 2D action platform game. Players control a character in a pixelated world, fighting enemies and navigating through various levels with different challenges and power-ups.

Game Projects

"The experiences gained in this project have led to a significant improvement in areas of algorithmic thinking, memory management, and fundamental game development skills. The 'So Long' project involved the development of a simple 2D game using the C programming language. Through this project, I had to enhance my memory management skills as I needed to organize the game world, characters, and objects efficiently in memory. Additionally, I had to effectively utilize data structures and algorithms to execute the project, which aided in making my code more efficient and optimized. The challenge of the project encompassed integrating various game components and enabling user interaction. It required implementing various input controls and screen updates to provide opportunities for the user to comprehend the flow of the game and engage with it. At this stage, a meticulous approach was necessary to ensure the proper functioning of the program and achieve the expected outcomes at each step. The 'So Long' project not only honed my technical skills but also improved my time management, troubleshooting, and collaboration abilities. Throughout the project, I had the chance to employ research and problem-solving skills to tackle challenges encountered. Moreover, I developed my collaboration skills by exchanging ideas and assisting fellow students, thereby enhancing our collective ability to complete the project successfully."
